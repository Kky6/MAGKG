# MAGKG KG Provenance Subset

This directory contains a KG subset for demonstrating evidence-preserving graph construction.

- `kg_trace_chunks.jsonl`: chunk-level input rows with typed entities, aliases, schema-valid relations, and document/paragraph/sentence/chunk provenance.
- `kg_trace_triples.jsonl`: flattened triples with evidence text, chunk text, and paragraph text.
- `canonical_graph_subset.json`: canonical graph generated from the trace chunks by `scripts/run_demo.py`.
- `evidence_trace.jsonl`: edge-level evidence table generated by `scripts/export_evidence_table.py`.
- `normalization_disambiguation_notes.json`: representative normalization and disambiguation notes from the canonicalization stage.
